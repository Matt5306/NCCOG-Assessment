🔒 Primary Findings — Infrastructure Security

Critical security issues identified during on-site assessment.

Gateway Using Factory Default Credentials Critical
The Calix gateway uses factory default credentials that are publicly documented. Anyone with network access can log into the admin interface, modify settings, monitor traffic, or disable security.
Affected
192.168.1.1 (Calix Gateway)
No Network Segmentation High
All devices share a single flat network (192.168.1.0/24). No VLANs configured. Staff computers, guest phones, IoT devices, and media equipment can all communicate freely.
Guest WiFi Not Isolated High
Despite having "NCC Guest" SSIDs, these networks are not isolated. Guest devices can access staff computers, printers, and the gateway admin interface.
IoT Device Exposure High
Donated Android monitors, thermostats, and video equipment have full network access. These devices don't receive security updates and represent attack entry points.
Infrastructure Capacity Medium
24-port switch at capacity with damaged ports. WiFi coverage inadequate for 64,000 sq ft — needs 20-24 access points vs current 3-4 consumer APs.
