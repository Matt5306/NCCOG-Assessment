Primary Findings — Infrastructure Security

Critical Gateway Using Factory Default Credentials

The Calix gateway/router is configured with factory default credentials that are publicly documented online. Anyone with network access can log into the gateway admin interface, modify settings, monitor traffic, or disable security features.

High No Network Segmentation

All devices share a single flat network (192.168.1.0/24). Staff computers, guest phones, IoT devices, and media equipment can all communicate with each other.

High Guest WiFi Not Isolated

Despite having guest SSIDs, these networks are not properly isolated from staff resources. Guest devices can access staff computers, printers, and the gateway admin interface.

High IoT Device Exposure

Donated display monitors running older Android operating systems have full network access. These devices no longer receive security updates and represent potential attack entry points.

Medium Infrastructure Capacity Issues

The 24-port switch is at full capacity with several damaged ports. WiFi coverage is inadequate for the 64,000 sq ft facility.

Secondary Findings — Network Scan

31 active hosts identified. See CF-2 for details.
